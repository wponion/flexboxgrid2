flexboxgrid2

A grid system based on the flex display property.

Github

Responsive

Responsive modifiers enable specifying different column sizes, offsets, alignment and distribution at xs, sm, md, lg & xl viewport widths.

<div class="wpo-row">
    <div class="wpo-col-xs-12
                wpo-col-sm-8
                wpo-col-md-6
                wpo-col-lg-4
                wpo-col-xl-2">
        <div>Responsive</div>
    </div>
</div>

Fluid

Percent based widths allow fluid resizing of columns and rows.

.wpo-col-xs-6 {
  flex-basis: 50%;
}

Simple Syntax

All you need to remember is row, column, content.

<div class="wpo-row">
    <div class="wpo-col-xs-12">
        <div>12</div>
    </div>
</div>

Offsets

Offset a column

<div class="wpo-row">
    <div class="wpo-col-xs-offset-3 wpo-col-xs-9">
        <div>offset</div>
    </div>
</div>

Auto Width

Add any number of auto sizing columns to a row. Let the grid figure it out.

<div class="wpo-row">
    <div class="wpo-col-xs">
        <div>auto</div>
    </div>
</div>

Nested Grids

Nest grids inside grids inside grids.

<div class="wpo-row">
    <div class="wpo-col-xs">
        <div>
            <div class="wpo-row">
                <div class="wpo-col-xs">
                    <div>auto</div>
                </div>
            </div>
        </div>
    </div>
</div>
